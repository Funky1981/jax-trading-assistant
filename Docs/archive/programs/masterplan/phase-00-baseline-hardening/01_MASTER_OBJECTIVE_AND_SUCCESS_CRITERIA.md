# Master Objective and Success Criteria

## Objective
Build a robust Jax platform that can research, paper trade, and eventually shadow/live trade IBM-style event repricing strategies with full auditability and no fake data contamination.

## Success criteria
- Event -> symbol mapping -> strategy -> risk -> order -> fill -> P/L traceable end-to-end
- UI-configurable strategy instances (no code changes per instance)
- Real deterministic backtests only for research conclusions
- Research/Analysis/Testing pages fully integrated
- Flatten-by-close and reconciliations working
- Trust gates pass consistently
