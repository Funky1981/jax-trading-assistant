# UI Master Plan

## Routes
- /research
- /analysis
- /testing
- /trading
- /system

## Research
Strategy instances, backtest launcher, sweeps/walk-forward projects, artifacts, provenance badges.

## Analysis
Run explorer, metrics/trades, decision timeline, AI decision inspector, provenance and dataset details, exports.

## Testing
Gate dashboard/history, test triggers, proof artifacts, incident panel.

## Trading integration
Event-tagged signals, instance IDs, AI advisory badge, risk reasons, flatten status, link to analysis timeline.

## Anti-fake-data UI rule
Synthetic/demo data must be hard-labeled and isolated. No promotion actions.
