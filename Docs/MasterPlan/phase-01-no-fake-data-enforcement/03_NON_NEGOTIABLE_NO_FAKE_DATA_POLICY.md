# No Fake Data Policy (Non-Negotiable)

## Rule
Zero fake/synthetic/generated market/backtest/trade data in any path used for:
- paper trading
- research conclusions
- analysis pages
- trust gates
- strategy promotion

## Enforcement
- runtime mode policy (dev/test/research/paper/live)
- startup provider validation blocks fake providers in paper/live
- provenance fields on runs/artifacts
- provenance trust gate
- UI badges for provenance (REAL / SYNTHETIC / UNKNOWN)
- artifact promotion blocked if evidence is synthetic or unknown
