# Jax Master Everything Plan v1 (Work Branch)

Single master plan for Jax to become a robust, auditable, no-fake-data, research-first event trading platform.

Includes:
- IBM-style event trading
- Research / Analysis / Testing pages and operations
- Strategy types + UI-configurable instances
- AI advisory + audit/replay
- Trust gates + shadow validation
- Execution hardening
- No-fake-data enforcement
- DB/API/UI/ops/observability/runbooks

Non-negotiable:
1. No fake data in paper/live paths
2. No fabricated backtests used for decisions
3. AI advisory only
4. Everything auditable/replayable
5. No gate pass, no trading
